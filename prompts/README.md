# Prompt Engines

This directory documents the third-party prompt engines used as internal
powering components inside Project-G-Live modules.

All prompts are from Steve Little's Open-Genealogy project:
https://github.com/DigitalArchivst/Open-Genealogy

License: CC BY-NC-SA 4.0
Used here for personal non-commercial research only.

## Assignments

| Fact Extractor v4 | 05 Document Analysis Worksheet | Extracts discrete factual claims from transcription |
