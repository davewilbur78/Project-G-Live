THE BUG (preserved for future reference)

Symptom: /investigation renders as unstyled text on parchment-dark background,
  no interface chrome, "no other modules."

Root cause: .next dev cache from a long-running dev server becomes stale when new
  routes / new CSS sources are added by another session. The CSS asset hash in the
  served HTML no longer matches any file on disk -- the stylesheet 404s and the
  page collapses to raw HTML. The same staleness can cause routes added since
  dev-start to 404.

Fix: kill the stale dev server, rm -rf .next, npm run dev. Verified in this session.
