# Module 04: Citation Builder

> See [AGENT.md](../../AGENT.md) for all operating rules.

## Purpose

Evidence Explained-style citations for every source type encountered
in genealogical research. The foundation every other module depends on.

## Status

NOT STARTED -- BUILD THIS FIRST. This is the first module to build
in Phase 3. No other module can function without it.

## Description

The Citation Builder generates fully formed citations in Evidence
Explained (EE) style for any source type encountered in genealogical
research -- vital records, census records, land records, probate records,
newspapers, photographs, databases, websites, and more. For each citation
the module collects the required fields through a structured interview,
classifies the source as Original, Derivative, or Authored, identifies
whether the information is Primary or Secondary based on the informant's
knowledge, and assigns an evidence classification of Direct, Indirect, or
Negative. It generates both a full citation and a short footnote form.
Both are required before a source may be cited in formal output. The
Image Citation Builder v2 prompt engine powers citation generation for
uploaded document images.

## Key Inputs

- Source metadata collected through structured interview (repository,
  title, author, date, URL, page, etc.)
- Source type (vital record, census, passenger manifest, naturalization,
  land record, probate, newspaper, photograph, database, website, etc.)
- Uploaded document image (optional -- triggers image citation path)

## Key Outputs

- Full Evidence Explained-style citation (first-reference form)
- Short reference note (subsequent-reference footnote form)
- Source classification: Original, Derivative, or Authored
- Information classification: Primary or Secondary (based on informant's
  knowledge)
- Evidence classification: Direct, Indirect, or Negative

## GPS Touchpoints

- Directly enforces complete and accurate citations (GPS element 2)
- Applies GPS source and information classification terminology throughout
- Prevents non-GPS terms ("primary source," "secondary source") from
  appearing anywhere in the platform
- Evidence classification (Direct/Indirect/Negative) feeds into proof
  argument construction in Case Study Builder (10)

## Prompt Engines Used

- **Image Citation Builder v2** (Steve Little) -- generates citations for
  uploaded document images
- **GRA v8.5c** -- GPS enforcement layer applied across all output

## Data Written to Supabase

- `sources` -- source records with full citation, short footnote, source
  type, information type, and evidence type classification
- `citations` -- citation records linking specific facts to sources

## Connection to Other Modules

- Citations are consumed by every other module in the platform
- Source classifications feed Document Analysis Worksheet (05) and
  Source Conflict Resolver (06)
- Citations appear in Timeline Builder (07), Family Group Sheet Builder
  (11), Research Report Writer (09), and Case Study Builder (10)
- File names for uploaded documents are generated by File Naming System
  (13) using citation metadata
- DNA Evidence Tracker (14) links match hypotheses to citations produced
  here
